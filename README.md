# Python Projects

## Opis
Repozytorium zawiera projekty oraz zadania wykonane w języku Python w trakcie nauki i ćwiczeń.

Głównym projektem w repozytorium jest implementacja systemu predykcyjnego T9, symulującego sposób wprowadzania tekstu znany z dawnych telefonów komórkowych.

---

## Główny projekt: T9 Predictive Text System

Aplikacja desktopowa stworzona w Pythonie, która symuluje działanie systemu T9 (predictive text), umożliwiającego przewidywanie słów na podstawie wprowadzanych cyfr.

### Funkcjonalności
- wprowadzanie tekstu za pomocą klawiatury numerycznej pokazanej na ekranie
- predykcja słów na podstawie sekwencji cyfr
- prosty interfejs graficzny

### Technologie
- Python
- Tkinter
